## Web 이란?
### 웹의 시초
: https://info.cern.ch/
- 처음 만들어진 웹은 전 세계 사람들이 쉽게 정보를 공유하고 접근하게 하기 위해 태어남
- 글자밖에 없었음
- **WWW(World Wide Web)**: 웹의 정식 명칭이며, 보통 Web이라 줄여 부름

### HTML
- **HTML(Hyper Text Markup Language)**: 초능력이 있는 텍스트로 구조를 나타내는 언어
  - 초능력이 있는 텍스트: 링크
  -> 즉, 초능력(링크 기능)을 부여하고 문서의 구조(제목, 본문 등)를 정의하는 데 사용되는 웹의 기본 마크업 언어
  - ex)
```
				<h1>안녕하세요</h1>
				<p>안녕하세요</p>
```

>**웹이란 결국, 우리 모두를 위한 정보 공유의 공간**

## Web vs Internet
- **인터넷(Internet)**: 전세계의 컴퓨터과 네트워크가 서로 연결되어 있는 거대한 통신망. 전세계 사람들이 접근 가능
- 웹은 인터넷상의 공간 ->인터넷이라는 거대한 통신 네트워크에 어느 한 지점을 집어 정보를 공유하는 공간을 만든 것이 웹
- 인터넷은 다른 세상, 즉 약속을 지켜야 함 -> HTTP

## HTTP
- **HTTP(Hyper Text Transfer Protocol)**: 웹 환경에서 클라이언트(브라우저)와 서버가 서로 통신하기 위해 따르는 '약속' 또는 '규약'
  - 인터넷 세상을 돌아다니기 위해 필요한 약속
  - 인터넷 상에서 데이터를 요청하고 응답받는 통신 프로토콜
    - 프로토콜 = 약속/규약
  - 쉽게 말해 브라우저가 서버에 페이지를 요청하고, 서버가 그 결과를 보내주는 약속된 방식
  - ex.) http://naver.com
